Ejercicios(5.a) y (6.a)

¿qué sucede con x=-1 ?

La función f5a no está definida en x=-1. Su evaluación en este punto dará el valor Inf, pero plot ignora este valor y dibuja la gráfica usando el resto de los puntos.

f5a = @(x) sqrt((1-x)./(1+x));
x = linspace(-1,1);
plot(x, f5a(x))

Ejercicios (5,b) y (6.b)

¿qué sucede con x=0?

La función f5b no está definida para x <= 0 ni para x = 1. Evitamos el punto x = 1. Observa lo que pasaría si dibujas una sóla curva en el intervalo (0,5)

f5b = @(x) x./log(x);
x1 = linspace(0, 1);
x2 = linspace(1, 5);
plot(x1, f5b(x1), x2, f5b(x2) )

Ejercicio (8.d)

La función f está definida para x < -sqrt(27) y para x > \sqrt(27). Dibujamos sus dos ramas por separado, pero ambas con el mismo color, ya que corresponden a la misma función.

La orden legend se usa aquí de forma algo distinta: identificamos cada curva con un número (h1, h2, h3) y luego se asigna leyenda sólo a las que se desee.

clf                            % para limpiar la ventana gráfica
a  = sqrt(27);
x  = linspace(-15, 20);
x1 = linspace(-15, -a);
x2 = linspace(a, 20);
f  = @(x) log(x.^2 - 27);
g  = @(x) 0.01*(x+10).*(x-10).*(18-x);
axis([ -15, 20, -20,15])
hold on
h1 = plot(x1, f(x1), 'b', 'LineWidth', 1.1);
h2 = plot(x2, f(x2), 'b', 'LineWidth', 1.1);
h3 = plot(x, g(x),   'r', 'LineWidth', 1.1);
legend([h1, h3], 'f(x)', 'g(x)')
grid on
hold off
shg

Ejercicio 9

function [N] = Multiplo(k, h)
%---------------------------------------------------------------
%  N = Multiplo(k, h)  devuelve
%         N = 1    si k+h es multiplo de 2
%         N = 2    si además es múltiplo de 3
%         N = 0    en otro caso
%---------------------------------------------------------------
N = 0;
z = k+h;
if mod(z,2) == 0
    N = 1;
    if mod(z,3) == 0
        N = 2;
    end
end

function [N] = Localizar(x,y)
%----------------------------------------------------------
%  N = Localizar(x,y) devuelve
%      N = 1 si el punto (x,y) esta (estrictamente) dentro
%          del círculo de centro (1,1) y radio 1
%      N = 2 si, además, el punto (x,y) está estrictamente
%          dentro del círculo de centro (0,0) y radio 1
%      N = 0 en otro caso
%----------------------------------------------------------
N = 0;
if ( (x-1)^2+(y-1)^2 < 1 )
    N=1;
    if ( x^2+y^2 < 1 )
        N = 2;
    end
end

Ejercicio 12

Atención: esta función es distinta de la que se propuso en la hoja de ejercicios, que se llamaba Rectas

function [A] = Areatri(a, b, c)
%----------------------------------------------------------
%  A = Areatri(a,b,c) es el area del triangulo de
%            lados a, b, c
%----------------------------------------------------------
if ( a < 0 || b < 0 || c < 0 )
    error('Las longitudes no pueden ser negativas.')
end
p = (a + b + c)/2;
A = p*(p-a)*(p-b)*(p-c);
if A < 0
    error('Las longitudes son incompatibles.')
end
A=sqrt(A);

function [vnorm] = Norma(v)
%----------------------------------------------------------
%   Norma(v) es la norma euclídea del vector v
%----------------------------------------------------------
vnorm = 0;
for k = 1:length(v)
    vnorm = vnorm + v(k)^2;
end
vnorm = sqrt(vnorm);

Ejercicio (18.b)

function [vmed] = Media(v)
%----------------------------------------------------------
%   Media(v) es la media aritmetica de los elementos
%            del vector v
%----------------------------------------------------------
vmed = 0;
n = length(v);
for k = 1:n
    vmed = vmed + v(k);
end
vmed = vmed/n;

function [u] = ProdAb(A, b, n)
%----------------------------------------------------------
%   u = ProdAb(A, b, n) es el producto de
%           A : matriz cuadrada nxn
%           b : vector de longitud n
%----------------------------------------------------------
u = zeros(size(b));
for i = 1:n
    sum = 0;
    for j = i:n
        sum = sum + A(i,j) * b(j);
    end
    u(i) = sum;
end

Ejercicio 22

Construir la intersección de dos conjuntos.
La función de MATLAB *intersect(x,y)* hace lo mismo, pero
en el caso general, sin suponer que no hay elementos repetidos
function [w]=Inter(x,y)
%----------------------------------------------------------
%  w = Inter(x,y)  es la intersección de los conjuntos x e y
%      Se supone que ni x ni y contienen elementos repetidos
%----------------------------------------------------------
w = [];
for k=1:length(x)
  z = x(k);
  for h=1:length(y)
    if ( z==y(h) )
      w = [w,z];
      break
    end
  end
end
